## 📖 Introduction

Muslimify is a Web Based Quran Application.

## 🖥️ Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Muslimify Api](https://github.com/gufronnakaaw/muslimify-api)

## 🌟 Features

- Dark Mode
- List Surah
- Random Quote in Homepage
- Daily Doa
- Asmaul Husna
